push as new branch:
----------------------------------------------------------------------------------------------------------
git switch -c v5    # makes the branch right where you are
git add -A
git commit -m "Version 5 – add full project snapshot"
git push -u origin v5

----------------------------------------------------------------------------------------------------------
push to same branch:
----------------------------------------------------------------------------------------------------------
git add -A && git commit -m "WHAT you changed, and WHY" && git push

or:

| Step                             | Command                                     | What it does                                                                                                       |
| -------------------------------- | ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| 1️⃣ Make sure you’re **on** `v5` | `bash git switch v5 `                       | (If you’re already there Git will say “Already on ‘v5’”).                                                          |
| 2️⃣ See what changed             | `bash git status `                          | Red = modified/ deleted • green = new files                                                                        |
| 3️⃣ Stage the changes            | `bash git add -A `                          | `-A` = add **all** new / changed / deleted files. (To add just one file use `git add file.txt`.)                   |
| 4️⃣ Commit them                  | `bash git commit -m "Describe the change" ` | Creates a new checkpoint in the `v5` history.                                                                      |
| 5️⃣ Push upstream                | `bash git push `                            | Because you did `git push -u origin v5` the first time, plain `git push` now knows to send commits to `origin/v5`. |



------------------------------------------------------------------------------------------------------------
pull:
------------------------------------------------------------------------------------------------------------
git switch v5
git pull --rebase    # recommended


# Start of day / before new work
git switch v5
git pull --rebase        # get everyone else’s work

# …make edits…
git add -A
git commit -m "Your changes"
git push                 # send your new commit up

--------------------------------------------------------------------------------------------------------------
create new folder and connect it (pull) to a branch:
--------------------------------------------------------------------------------------------------------------
git remote add origin https://github.com/hoomanka/itrc_eval_system.git
git fetch origin --prune --tags
git switch -c v6 origin/v6         # modern Git
#  └─ older Git      git checkout -b v6 origin/v6
